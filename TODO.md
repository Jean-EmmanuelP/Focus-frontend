## Angle d'attaque — Stratégie de vente

### Positionnement
> "Ton tel travaille contre toi. Focus met un pote exigeant dedans."

Pas une app de to-do. Pas un timer. Un coach IA qui te connaît, voit que tu scrolles, et te dit d'arrêter.

### Cible principale
- Étudiants et jeunes actifs (18-28 ans)
- Ceux qui galèrent avec le scroll compulsif
- Période clé : révisions, rentrée, examens

### Contenu organique (TikTok / Reels / X)
- [ ] Filmer la notif de distraction en temps réel (scroll → "Hey 👀 Tu scrolles là") — contenu viral
- [ ] Contenu "avant/après" screen time (montrer son propre screen time qui baisse)
- [ ] Montrer des conversations avec le coach (le ton "pote exigeant")
- [ ] Format "1 semaine avec Focus" — transformation visible
- [ ] Posts punchlines : "J'ai mis un coach dans mon tel pour m'engueuler"

### Communautés à infiltrer
- [ ] Discord serveurs étudiants / révisions
- [ ] Reddit : r/getdisciplined, r/productivity, r/studytips
- [ ] Groupes Facebook révisions / concours / prépa
- [ ] Twitter/X : communauté dev, indie hackers, #BuildInPublic

### Différenciation vs concurrence (Forest, Flora, Focus To-Do)
- Coach IA qui parle comme un pote (pas un robot froid)
- Notifications de distraction EN TEMPS RÉEL pendant le scroll
- Blocage d'apps avec un coach qui refuse de débloquer ("Non. 5 min ça devient 1h")
- Félicitations quand tu coches une tâche → engagement émotionnel

### Modèle freemium
- **Gratuit** : Chat + blocage d'apps + tâches/routines + notifs distraction
- **Pro** : Vocaux illimités, stats avancées, personnalisation du coach, appels vocaux

### App Store Optimization (ASO)
- [ ] Titre : "Focus — Coach Anti-Scroll" ou "Focus — Ton Coach IA"
- [ ] Subtitle : "Il te sort du scroll."
- [ ] Keywords : screen time, bloquer apps, productivité, concentration, coach IA, focus, anti scroll, habitudes
- [ ] Description : axer sur le problème (scroll) → la solution (un pote dans ton tel)
